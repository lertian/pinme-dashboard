# 实施计划 - 最小 MVP 版本

本文档包含实现用户项目管理功能的**最小可行产品（MVP）**任务列表。专注于核心功能，快速交付可用版本。

## MVP 核心功能

✅ 用户注册和登录（用户名/密码）  
✅ 项目列表展示（卡片式）  
✅ 上传新项目  
✅ 更新项目版本  
✅ 管理员查看用户和重置密码  
✅ 用户修改自己的密码  
✅ 上传进度条展示

## MVP 功能范围总结

1. **基础认证**：本地 SQLite 存储，JWT 安全令牌。
2. **项目列表**：精美卡片展示，支持搜索、更新及删除。
3. **上传/部署**：自动解压/移动文件夹，本地服务器直接预览。
4. **实时进度**：上传大文件时提供视觉进度条反馈。
5. **管理台**：管理员可掌控用户状态。

现在您可以直接运行 `npm run start-app` 体验完整功能！
